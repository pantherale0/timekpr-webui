# Screenshot assets

Store UI screenshots for documentation under `docs/assets/images/`.

## Guidelines

1. Capture from a running Guardian dev or staging server
2. Save as PNG (recommended width ~1920px)
3. Reference in markdown: `![Dashboard](../assets/images/web-ui/dashboard.png)`

## Suggested captures

- Dashboard user cards
- Weekly schedule editor
- Time adjust modal
- Device detail (Android policy tab)
- Add Device wizard
