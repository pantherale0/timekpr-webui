Unlocking Advanced Restrictions
TimeKpr needs Device Owner (DO) permission for these controls.

Enforcing hardware limitations, preventing uninstall, and automatically provisioning child user accounts requires the TimeKpr agent to be set as Device Owner.

To provision via ADB:
  1. Enable Developer Options and USB Debugging on the child's device.
  2. Remove all existing user accounts from the device under Settings → Passwords & Accounts.
  3. Connect the device to a computer and execute the following terminal command:
adb shell dpm set-device-owner com.timekpr.agent/.admin.TimeKprDeviceAdminReceiver

Once successfully executed, restart the TimeKpr agent. The Device Owner status will refresh and unlock these restrictions.
