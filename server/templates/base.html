On the Linux computer, open a terminal and run this one-line installer to connect Guardian:

curl -sSL http://yourserver/scripts/install-agent.sh | sudo bash -s -- --url ws://yourserver/ws

Guardian installs quietly in the background and starts a secure pairing handshake with this console.

Download the Guardian APK, open it on the child's Android device, and scan the QR code to connect:

MDM provisioning is unavailable. Make sure an Android APK has been uploaded in Settings.

On the Windows computer, open PowerShell as Administrator and paste this command to install Guardian:

powershell -ExecutionPolicy Bypass -Command "..."

Alternatively, you can download the MSI installer package directly and run it:

Download MSI Installer

The installer registers the background monitoring service and configures autostart components for user sessions.
