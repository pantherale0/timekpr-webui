Open a terminal window on the target Linux machine and execute the following installation script command to install the Guardian Agent Daemon:

curl -sSL http://yourserver/scripts/install-agent.sh | sudo bash -s -- --url ws://yourserver/ws

The script registers a systemd background service and initiates the secure pairing process.
